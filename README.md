# AppGlobal

Global code for any Android App.

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.jenilgopani6969:AppGlobal:1.1.2'
	}
